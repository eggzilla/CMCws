Matrix of link scores:
The matrix considers all submitted models.
Color-code emphasizes high scoring pairs:
  • link score >20: red
  • link score <20 >10: orange
  • link score <10 >5: yellow
  • link score <5 >0: white
  • link score <0: blue
--> For more information see Help
Covariance model graph
The graph contains the models from the list below
and shows their link scores. If the list is manipulated
with the filtering options, this changes the graph.
--> For more information see Help
Filtering:
The list of models below can be filtered or expanded with these options.
  • Number of hits displayed even
    if more would fulfill the other criteria.
    Only numbers allowed. Default is 10.
  • Cutoff sets the minimal linkscore for hits to be displayed. Only numbers allowed. Not set per default (none).
  • Model name, if set, shows only hits containing this string. Only numbers or letters allowed. Not set per default (none).
--> For more information see Help
Detailed Comparison:
Submits the selected models and the input model
for comparison with each other
--> For more information see Help
Result Table:
Sort the list by clicking the header of the column to be sorted.
Magnifing glasses link to a detailed view of each comparison.
CMCompare computes a score for the input model (Input score) and one for the Rfam model (Rfam score). The lower value is the Link score.

Optionally a set of interesting models can be compared versus each other by selecting the checkboxes at the beginning of those lines and clicking cm vs cm.
--> For more information see Help
